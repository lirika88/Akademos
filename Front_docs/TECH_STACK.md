## Технологии, используемые в проекте

Для реализации клиентской части приложения "Akademos" используется следующий стек технологий:

| Категория | Технология | Обоснование |
| :--- | :--- | :--- |
| **Ядро** | HTML5 | Для создания структуры веб-страниц (разметка заголовков, списков, форм, таблиц). |
| **Стилизация** | CSS3 | Для оформления внешнего вида: цвета, шрифты, отступы, адаптивная верстка с использованием Flexbox и Grid. |
| **Язык программирования** | JavaScript (ES6+) | Для реализации интерактивности: обработка кликов, отправка форм, динамическое обновление контента без перезагрузки страницы. |
| **Работа с API** | Fetch API | Встроенный в браузер метод для отправки HTTP-запросов к серверу и получения данных в формате JSON. |
| **Хранение данных на клиенте** | LocalStorage | Для временного хранения токена авторизации и небольших пользовательских настроек. |
| **Иконки** | Font Awesome | Библиотека векторных иконок для визуального оформления интерфейса (кнопки, меню, статусы). |
| **Контроль версий** | Git / GitHub | Для отслеживания изменений в коде, совместной разработки и хранения репозитория. |
| **Деплой (хостинг)** | Netlify | Для автоматической публикации статического сайта. При пуше в ветку `main` сайт обновляется автоматически. |

**Выбор технологий** обусловлен тем, что проект представляет собой статическое веб-приложение (фронтенд), не требующее сложных фреймворков. HTML, CSS и JavaScript являются стандартными технологиями веб-разработки, поддерживаемыми всеми браузерами. Использование Fetch API позволяет эффективно взаимодействовать с серверной частью, разрабатываемой вторым участником команды. Для хостинга выбран Netlify как наиболее простой и удобный сервис для развертывания статических сайтов с автоматическим обновлением при изменениях в репозитории.
