# API документация

## Базовый URL

В разработке: `http://localhost:3000/api`  
На production: будет заменён на реальный домен.

## Аутентификация

Большинство эндпоинтов требуют JWT-токен, который передаётся в заголовке:

Токен получается при успешном входе (см. эндпоинт `/auth/login`).

## Основные эндпоинты

### Аутентификация

| Метод | Эндпоинт           | Описание                         | Тело запроса (JSON)                                      | Ответ (успех)                          |
|-------|--------------------|----------------------------------|----------------------------------------------------------|-----------------------------------------|
| POST  | `/auth/register`   | Регистрация нового пользователя | `{ "email": "...", "password": "...", "role": "tutor" }` | `{ "message": "User created", "userId": 1 }` |
| POST  | `/auth/login`      | Вход в систему                   | `{ "email": "...", "password": "..." }`                  | `{ "token": "jwt_token", "user": { ... } }` |
| GET   | `/auth/me`         | Получить информацию о текущем пользователе (требуется токен) | - | `{ "id": 1, "email": "...", "role": "tutor" }` |

### Ученики

| Метод | Эндпоинт               | Описание                          | Параметры запроса              | Ответ (успех)                          |
|-------|------------------------|-----------------------------------|--------------------------------|-----------------------------------------|
| GET   | `/students`            | Список учеников репетитора        | `?search=имя&page=1&limit=10` | `{ "students": [...], "total": 100 }`  |
| POST  | `/students`            | Добавить нового ученика           | `{ "name": "...", "email": "..." }` | `{ "student": { ... } }`          |
| GET   | `/students/:id`        | Получить карточку ученика         | -                              | `{ "student": { ... } }`               |
| PUT   | `/students/:id`        | Обновить данные ученика           | `{ "name": "..." }`            | `{ "student": { ... } }`               |
| DELETE| `/students/:id`        | Удалить ученика                   | -                              | `{ "message": "Student deleted" }`     |

### Материалы

| Метод | Эндпоинт               | Описание                          | Примечание                               |
|-------|------------------------|-----------------------------------|------------------------------------------|
| POST  | `/materials/upload`    | Загрузить файл                    | `multipart/form-data` с полем `file`    |
| GET   | `/materials`           | Получить список файлов/папок      | Поддерживает параметры `folderId`        |
| GET   | `/materials/:id`       | Скачать или получить информацию о файле | Возвращает файл или JSON         |
| DELETE| `/materials/:id`       | Удалить файл/папку                | -                                        |

### Тесты и задания

| Метод | Эндпоинт               | Описание                          | Тело запроса                              |
|-------|------------------------|-----------------------------------|--------------------------------------------|
| POST  | `/tests`               | Создать тест                      | `{ "title": "...", "questions": [...] }` |
| GET   | `/tests`               | Список тестов репетитора          | -                                          |
| GET   | `/tests/:id`           | Получить тест с вопросами         | -                                          |
| POST  | `/tests/:id/assign`    | Назначить тест ученику            | `{ "studentId": 1, "deadline": "..." }`  |
| POST  | `/tests/:id/submit`    | Отправить ответы на тест (ученик) | `{ "answers": [...] }`                    |
| GET   | `/results/student/:id` | Получить результаты ученика       | -                                          |

### Занятия

| Метод | Эндпоинт               | Описание                          | Тело запроса                              |
|-------|------------------------|-----------------------------------|--------------------------------------------|
| GET   | `/lessons`             | Получить занятия (с фильтром по дате) | `?start=2023-01-01&end=2023-01-31` |
| POST  | `/lessons`             | Создать занятие                   | `{ "studentId": 1, "date": "...", "topic": "..." }` |
| PUT   | `/lessons/:id`         | Обновить занятие                  | `{ "topic": "..." }`                      |
| DELETE| `/lessons/:id`         | Удалить занятие                   | -                                          |
